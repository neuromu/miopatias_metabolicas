# 🧬 MIOPATIAS METABÓLICAS - VERSÃO COMPACTA E OBJETIVA

## 📌 Introdução Rápida

**Miopatias metabólicas** = déficits genéticos no metabolismo energético muscular

**Apresentação:** Crises de fraqueza, dor e/ou rabdomiólise (CK >10x normal) durante/após exercício

**Exame neurológico:** Geralmente normal entre eventos
